# Year 9 Maths Question Imports

Generated by [`_generate.py`](_generate.py) from a set of Year 9 maths exam and
worksheet PDFs (supplied 2026-06-11). One `<topic>.json` **and** one matching
`<topic>.zip` per topic; each ZIP contains a single file `questions.json` with
identical content. The format matches `MathsQuestionParser` in
`cwa_classroom/classroom/upload_services.py` (`subject: maths`,
`year_level: 9`, `strand` = parent topic, multiple choice, exactly one correct
answer per question, 1-based `order`).

Regenerate with: `python imports/year9/_generate.py`

Every answer was verified by computation; where the PDF had an answer key the
two were cross-checked and agreed (exceptions noted below). No distractor is
mathematically equal to its correct answer (enforced by the generator's
numeric-equality check plus manual review of symbolic options).

### Not convertible to single-answer multiple choice
Sketching/plotting/shading tasks (HY1 Q16, WS78 Q3-plot, WS80 Q2/Q5, WS81 Q2…),
"show that"/proof questions (HY1 Q17, Q19c; HY2 Q4a, Q6c-i, Q7a, Q7d, Q8c, Q8d),
explain/justify questions (HY1 Q24b, bike-path budget justification, Kris's-mistakes
explanation — its computable core was kept), table-completion drills, and
"write a possible set/equation" items with many acceptable answers
(the P ∪ Q set question was kept by recasting it as "which set satisfies…").

### Skipped for answer-key or ambiguity problems
- **Trig 4.3 Q3 (hot-air balloon)**: the workbook key says 235 m, but that is the
  balloon's *initial height*; the question asks for the distance to point P,
  which is ≈ 407 m. Skipped rather than ship a disputed answer.
- **Trig 4.1 Q19**: sin⁻¹(0.5628) = 34.25°, exactly on the 1-d.p. rounding
  boundary (key says 34.2°). Skipped as unsafe for auto-marking.
- **G9 Linear Equations Part 1 Ex14(a)**: table has unevenly spaced x-values;
  the "first differences" method the worksheet intends gives a different
  conclusion than direct rate calculation. Skipped as ambiguous.
- **G9 (1) P4-Q8**: side lengths "4/12, 5/12" may be mangled mixed numbers.

### Other notes
- The two "Half Yearly & Solutions" PDFs are different exams (zero overlapping
  questions); both were used.
- Trivial drill sets (e.g. twelve "find 15²" items) were sampled rather than
  imported one-for-one.
- Multi-part questions were split into self-contained single-answer questions,
  with any needed context restated in each question.
