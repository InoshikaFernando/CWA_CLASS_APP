# Year 8 Maths — Question Sets

Question banks in the upload format consumed by `MathsQuestionParser`
(`cwa_classroom/classroom/upload_services.py`), generated by
[`_generate.py`](./_generate.py).

Sources:

- **Batch 1** — "Year 8 Mid Year Exam" + "Theme 8 Revision" sheet.
- **Batch 2 (June 2026)** — "Algebra G8" linear-equations pack,
  "Year 8 Maths Exam Booklet — Algebra", "G8 Finance" pack,
  "G8 — Applications of Pythagoras Theorem", "Quantitative Reasoning Test"
  and the "Selective" practice exam.

## How to use

Upload either the `.json` directly, or the `.zip` (which contains a single
`questions.json` with the same content) via the classroom question-upload
screen. `year_level` is set to `8`, so a Level with `level_number = 8` must
exist.

All questions are `multiple_choice` with exactly one correct answer.
Distractors were chosen so none is mathematically equal to the correct answer
(no `1/2` next to `2/4`, no `1.130` next to `1.13`). Every numeric answer was
computed independently and cross-checked against the source's answer key where
one existed (the Pythagoras pack includes fully worked answer pages, which all
match). A few source questions shipped with *trap options* that are
mathematically equal to the correct answer (e.g. `√72` alongside `6√2`,
`3/36` alongside `1/12`, `(3x + 12)(x − 1)` alongside `3(x + 4)(x − 1)`); those
distractors were replaced with non-equivalent ones.
