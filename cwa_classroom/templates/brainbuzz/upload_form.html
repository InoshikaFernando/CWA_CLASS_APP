📋 Format Requirements

CSV Format

Required columns:

  • topic (topic name)
  • level (level number 1-8)
  • question_text
  • question_type (multiple_choice, true_false, short_answer, fill_blank)
  • difficulty (1, 2, or 3)
  • answer1, is_correct1 (for multiple choice/true-false)
  • correct_short_answer (for short answer/fill blank)

JSON Format

Example structure:

{
  "subject": "maths",
  "questions": [
    {
      "question_text": "What is 2+2?",
      "question_type": "multiple_choice",
      "difficulty": 1,
      "topic": "Arithmetic",
      "level": 3,
      "answers": [
        {"text": "4", "is_correct": true}
      ]
    }
  ]
}
