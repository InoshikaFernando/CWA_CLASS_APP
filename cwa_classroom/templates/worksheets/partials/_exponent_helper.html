{% comment %} Exponent input helper for algebra-graded questions. Renders a small toolbar above the answer field. The "x²" button inserts a "^" at the caret of the sibling text_answer input/textarea, so students can type powers without a special keyboard (e.g. 2x^2 - 7x - 15). Self-contained: the button finds its field via the DOM, so multiple questions on one page are fine. {% endcomment %}
Type powers with ^ — e.g. 2x^2 - 7x - 15. Combine like terms and expand brackets.
