{# ── Format reference — Mathematics ───────────────────────────────── #}

📄 JSON format (no images)

{
  "strand": "Number",
  "topic": "Fractions",
  "year_level": 4,
  "questions": [
    {
      "question_text": "What is 1/2 + 1/4?",
      "question_type": "multiple_choice",
      "difficulty": 1,
      "points": 1,
      "explanation": "Convert to a common denominator first.",
      "answers": [
        {"text": "3/4", "is_correct": true},
        {"text": "1/2", "is_correct": false},
        {"text": "2/6", "is_correct": false},
        {"text": "1/4", "is_correct": false}
      ]
    }
  ]
}

🖼️ ZIP format (with images)

my_upload.zip

├── questions.json

├── image1.png

└── ruler.jpg

Images are saved to media/questions/year<N>/<topic>/ automatically.

Duplicate detection: same question_text + topic + level → updated, not duplicated.

question_type: multiple_choice · true_false · short_answer · fill_blank · calculation

difficulty: 1 Easy · 2 Medium · 3 Hard

{# ── Format reference — Coding ─────────────────────────────────────── #}

💻 Coding exercise JSON format

{
  "subject": "coding",
  "language": "python",
  "topic": "loops",
  "level": "beginner",
  "exercises": [
    {
      "title": "For Loop: Print 1–5",
      "instructions": "Use a for loop to print numbers 1 through 5, each on a new line.",
      "starter_code": "# Use a for loop\nfor i in range(?, ?):\n    print(?)\n",
      "expected_output": "1\n2\n3\n4\n5",
      "hints": "range(1, 6) generates numbers 1 through 5",
      "display_order": 1
    },
    {
      "title": "While Loop: Countdown",
      "instructions": "Use a while loop to count down from 5 to 1.",
      "starter_code": "count = 5\nwhile count > 0:\n    # Print count and decrease it\n    pass\n",
      "expected_output": "5\n4\n3\n2\n1",
      "hints": "Don't forget: count -= 1 inside the loop",
      "display_order": 2
    }
  ]
}

Field reference

Field Required Notes
language✅slug: scratch, javascript, python, html, css
topic✅slug of an existing CodingTopic for that language
level✅beginner · intermediate · advanced
exercises[].title✅Duplicate title + topic + level → updated
exercises[].instructions✅Shown to the student
exercises[].expected_output—Exact stdout the correct solution produces (leave blank for HTML/CSS/DOM/Scratch)
exercises[].starter_code—Pre-filled editor code
exercises[].hints—Hint text shown on request
exercises[].display_order—Defaults to array index
{# ── Format reference — Coding Problems ───────────────────────────── #}

🧩 Coding problem JSON format

{
  "subject": "coding_problem",
  "language": "python",
  "problems": [
    {
      "title": "Reverse a String",
      "difficulty": 1,
      "category": "algorithm",
      "description": "Read a line and print it reversed.\n\nExample:\n  Input:  hello\n  Output: olleh",
      "starter_code": "s = input()\n# Print reversed\n",
      "constraints": "",
      "time_limit_seconds": 5,
      "memory_limit_mb": 256,
      "forbidden_code_patterns": [],
      "test_cases": [
        {"input": "hello", "expected": "olleh", "visible": true,  "boundary": false, "description": "Basic"},
        {"input": "a",     "expected": "a",     "visible": false, "boundary": true,  "description": "Single char"}
      ]
    }
  ]
}

Field reference

Field Required Notes
language✅slug: python · javascript
problems[].title✅Duplicate title + language → updated (test cases replaced)
problems[].description✅Full problem statement shown to the student
problems[].difficulty—1 (easiest) to 8 (hardest). Default: 1
problems[].category—algorithm · logic · data_structures · dynamic_programming · graph_theory · string_manipulation · mathematics · sorting_searching
problems[].starter_code—Skeleton code shown in the editor
problems[].constraints—Input size / value constraints
problems[].time_limit_seconds—Default: 5
problems[].memory_limit_mb—Default: 256
problems[].forbidden_code_patterns—Array of forbidden substrings, e.g. ["sorted(", ".sort("]
test_cases[].input✅stdin passed to the program
test_cases[].expected✅Expected stdout
test_cases[].visible—true = shown as sample to student. Default: false
test_cases[].boundary—true = edge/boundary test case. Default: false
test_cases[].description—Short label shown in admin drill-down
